# Python Journey

**Python Journey** is a learning project created during the training program conducted by **Hashtag Treinamentos**.

The goal of this repository is to document the progression through different Python topics by implementing small practical projects in each class.

Each lesson is organized in its **own subrepository folder**, allowing the learning process to be structured and easy to follow.

---

# 📚 Course Overview

The training is divided into **four main modules**, each focusing on a different application of Python.

## 1️⃣ Python Power Up

Focus on **automation and productivity**.

Topics covered:

* Task automation
* Bot creation
* Saving work hours through automation
* RPA and Web Scraping

---

## 2️⃣ Python Insights

Focus on **data analysis and extracting insights**.

Topics covered:

* Data analysis and data processing
* Working with tables (**DataFrames**)
* Creating charts and visualizations in Python
* Extracting meaningful insights from datasets

---

## 3️⃣ Python AI

Focus on **Artificial Intelligence and Data Science**.

Topics covered:

* Artificial Intelligence concepts
* Complete prediction project
* Data Science fundamentals
* Model creation and analysis

---

## 4️⃣ Python Dev

Focus on **software and application development with Python**.

Topics covered:

* Building websites and systems
* Python frameworks
* Chatbot development
* Intelligent responses using AI agents

---

# 📁 Repository Structure

Each class is implemented in its own folder:

```text
python-journey/
├── class-01/
├── class-02/
├── class-03/
└── class-04/
```

Each folder contains:

* The code developed during the class
* Small projects
* Experiments related to the topic
